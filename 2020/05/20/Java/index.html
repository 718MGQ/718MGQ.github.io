Java

volatile修饰

变量声明了volatile后，跳过cpu cache步骤，jvm保证每次直接从内存中读取变量

  • volatile声明的变量所有线程可见，即当修改一个变量值会立即更新直接更新内存，读取是也直接从内存中刷新读取
  • volatile声明的变量禁止指令重排序优化

volatile性能

volatile 的读性能消耗与普通变量几乎相同，但是写操作稍慢，因为它需要在本地代码中插入许多内存屏障指令来保证处理器不发生乱序执行
